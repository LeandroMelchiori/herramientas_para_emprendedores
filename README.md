# 🧮 Calculadora de Costos para Emprendedores

Una app web mobile-first para ayudar a emprendedores autogestivos a calcular el precio de venta de sus productos de forma simple y visual.

> App en uso real por emprendedores de la economia social de Santa Fe, Argentina.

🔗 **[Ver demo en vivo](https://tuusuario.github.io/calculadora-costos)**

---

## ✨ Funcionalidades

- **Cálculo de costos** — insumos, materiales y mano de obra por tanda de producción
- **Margen de ganancia** — slider interactivo o ingreso de precio manual con cálculo inverso
- **Punto de equilibrio** — cuántas unidades vender por mes/día para cubrir gastos fijos
- **Comparativa de márgenes** — tabla con distintos escenarios de precio
- **Proyectos guardados** — base de datos local con guardar, cargar y duplicar
- **Compartir por link** — estado completo codificado en la URL
- **Exportar / Importar** — backup de proyectos en formato JSON
- **Descarga en PDF** — resumen profesional del cálculo
- **Moneda configurable** — ARS, UYU, CLP, MXN, USD
- **Autoguardado** — nunca se pierden los datos al cerrar

---

## 🛠️ Tecnologías

- **HTML5 / CSS3 / JavaScript** — vanilla, sin frameworks ni dependencias
- **localStorage** — persistencia de datos en el navegador
- **jsPDF** — generación de PDF en el cliente
- **CSS Custom Properties** — sistema de diseño con variables
- **URL API** — codificación del estado para compartir por link
